📖 使用说明

  1. 打开微信 — 进入你想自动回复的群聊或私聊窗口
  2. 校准聊天区域 — 点击 可视校准，在微信截图上按住拖拽框选消息区域，绿色框内就是识别范围
  3. 配置 API Key — 在设置中填入你的 DeepSeek 或 Claude API Key
  4. 编辑个性文件 — 修改 personality/default.txt 来定制你的AI人设
  5. 启动 — 点击 启动 按钮，Agent 开始自动监控和回复
  6. 停止 — 随时点击 停止 或关闭本页面

💡 提示

  • 私聊模式：给好友创建 personality/friends/好友昵称.txt 可定制私聊风格
  • 群聊模式：自动识别多人群聊，只在@你或聊到兴趣话题时回复
  • 安全限制：默认 7:00-凌晨2:00 运行，每小时最多 120 条回复
  • 修改设置后需要 重新启动 Agent 才能生效
